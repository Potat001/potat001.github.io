3 måtar å legge inn CSS eigenskapar:
	• Inline
		○ <body style=""> for å legge til CSS eigenskapar i body-taggen. Kan gjere det med skriftelement òg, som <h1>.
			• style = : For å endre stilen på nettsida vår
				○ background color
				○ color
					§ rgb()
					§ Kan òg legge til #-verdiar for fargane
				○ ; etter fargen for å fortsette å endre fargen på noko anna du vil andre fargen på.
		
	• Internal
		○ <style>
	body{
		background-color: black;
	}
			• For å endre fargen på heile nettsida.
	• p{
		○ background-color: gray;
	color: white;
	}
			• For å endre fargen på paragrafane/<p>.
	• External (fungerer best)
		○ style.css
		○ For å linke html html-fila til css-style-fila vår:
			• <link rel="stylesheet" href="style.css">
		○ Kan gjenbruke CSS-style-en med å bruke den same linken som over.
			• Alle endringane i style.css vil gjelde for alle andre nettsider som bruker lenka over.
		○ Kan gi elementa forskjellege id-verdiar
			• <p id="p1"> og skriv inn 
			
			pi{
			    color:yellow;
			}
			i css.
		○ Kan lage ei klasse:
			• <p class="odd">
				.odd{
				color: red;
				}
		○ For å tildele ei klasse der alle funksjonane skal vere i.
		○ font-family: "consolas", sans-serif;
			• For å endre skrifttypen. Lister opp to i tilfelle nettsida ikkje støtter den skrifttypen.
			• fonts.google.com for å finne skrifttypar.
		○ For å endre tittelskrifttypen:
			<link href="https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap" rel="stylesheet"> i headeren i index.html
			font-family: "press Start 2P" i headeren i style.css.
		○ Diverse andre stilar:
			
			font-style: italic;
			    font-weight: bold;
			    text-decoration: green dotted underline;
			    font-size: 20px;
		
		• For å endre border/grensa på teksten vår:
			
			p{
			    border-style: ridged;
			}
		• Grensa på overskrifta vår:
			
			h1{
			    border-top-style: dotted;
			}
			
		• Diverse endre stilar:
		
		h1{
		    border-top-style: dotted;
		    border-bottom-style: dotted;
		    border-left-style:double;
		    border-left-width:10px;
		    border-left-color: silver;
		}
		p{
		    border-style: outset;
		    border-width: 10px;
		    border-color: gold;
		    border-radius: 10px;
		    padding: 100px;
		}
		• Bakgrunnar
			
			 background: linear-gradient(skyblue, lightgreen);
				• (to right, skyblue, lightgreen);, der fargane går mot/sluttar.
			    background-repeat: no-repeat;
			    background-attachment: fixed;
			• Siste: Sånn at bakgrunnen strekk seg heilt til enden av det vi kan sjå.
		• For tittelen, paragrafane og bildebrakgrunnen
			• 
			body{
			    background-image: url("Lihesten.jpg");
			    background-repeat: no-repeat;

			    background-repeat: no-repeat;
			    background-attachment: fixed;

		• Margin
			• Mellomrommet rundt eit element.
			• Sample tekst:
			
				<div id="box1">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Nihil error eius provident numquam cum necessitatibus libero rerum dicta ex magni possimus velit, laboriosam optio ducimus explicabo soluta beatae dolor cumque.</div>
				<div id="box2">Lorem ipsum dolor sit amet consectetur adipisicing elit. Placeat molestias rem voluptates quod maxime libero saepe consequatur aliquid architecto. Facilis minima nisi, neque tempora aut vero error ipsum qui sit?</div>
			
			• 
			#box1{
			    border: 5px solid;
			    width: 250px;
			    height: 250px;
			    font-size: 22px;
			    background: rgb(51, 255, 0);
			}
			#box2{
			    border: 5px solid;
			    width: 250px;
			    height: 250px;
			    font-size: 22px;
			    background: rgb(90, 27, 172);
			}
			• 
			body{
			    margin: 0px;
			}
				• Nå vil ikkje boksane ha noko mellomrom mellom enden av sida, og sjølve boksane.
				• Padding ikkje det same som margin.
			• 
			 margin-top: 50px;
			    margin-left: 50px;
			    margin-bottom: 50px;
			    margin-right: 50px;
			Eller:
			• 
			margin: 50px;
			• 
			margin-left: 50%;
				• Vil bruke 50% av breidda til vindauget vårt.
			eller bytte ut left med auto, for å dytte elementet til venstre automatisk.
			• 
			 margin-left: auto;
			    margin-right: auto;
			Vil sitte i midten heile tida.
	• Float
		• Flyttar element til den venstre eller høgre sida av sida.
		
		img{
		    float: left;
		}
		• 
		.box{
		    width: 100px;
		    height: 100px;
		    border: 1px solid;
		    font-size: 40px;
		    text-align: center;
		    background:tomato;
		    float: right;
		}
		p{
		    clear:both;
		}
		clear: both;: Vil flytte teksten frå teksten på begge sidene.
	• Positioning
		• 
		position: static;: Vil ikkje bevege seg.
		position:relative;: Kan bevege elementet i forhald til eit origo punkt.
			
			top: 150px;
			    left: 150px;
			Flyttar elementet.
		• Parent: Søker etter ein parent som ikkje er static.
			
			<div id="box1">
			        <div id="box2"></div>
			    </div>
			• Box1 ein parent av box2.
				
				#box1{
				    border: 1px solid;
				    height: 300px;
				    width: 300px;
				    background-color: turquoise;
				    position: relative;
				    left: 250px;
				    top: 250;
				• Kan nå flytte alle boksane frå box1.
		• Fixed: Gjer at elementet sit i same posisjonen, uansett om vi scrollar eller ikkje.
		
			position: fixed;
			    top: 0px;
			    right: 0px;
		• Sticky: Elementet vil halde seg fast til toppen/botnen av sida.
			
			position: sticky;
			    top: 0px;
			    bottom: 0px;
			
	• Pseudo klassar: Eit nøkkelord lagt til ein veljar som spesifiserer ein spesiel status til det spesifiserte elementet. Du kan klikke på dei, ha musa over dei, osv.
		• a: for å endre statusen til eit element.
		
		a:link{
		    color:lawngreen;
		}
		Der a er <a href="https://www.google.no">Google.com</a>
